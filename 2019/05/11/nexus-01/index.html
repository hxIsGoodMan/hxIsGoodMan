Nexus 3.x下载

免费版下载地址： https://www.sonatype.com/download-oss-sonatype

环境准备

我的环境如下：

  • jdk ： 1.8
  • maven ：3.6
  • 系统： centos 7

如果有想用 Windows 搭建私服的朋友， 直接闭着眼 跑就行了。 这里就不做详细解释了，哈哈哈~~

上干货！！

  • 下载好linux版的 nexus后，上传至 /usr/local 目录

进入 /usr/local 目录 -> 创建新文件夹 nexus -> 解压下载好的压缩包到 新创建的nexus文件夹中 -> 进入nexus文件夹 -> 重命名 nexus-3.6.0-02 为 nexus ；

  • 运行

启动需要一些时间，等到出现 👇👇👇

就是启动成功了 。

那么现在就可以访问 nexus了 ： http://172.16.104.203:8081

想改nexus 默认端口号的童鞋可以到：/usr/local/nexus/nexus/etc/nexus-default.properties 中修改的哦

nexus 创建私服仓库

先登录

  • 默认账号 ： admin
  • 密码 ： admin123

登录后 可以去新建一个账号 毕竟 admin 这样子的账号密码 和没有 没事没区别了 哈哈哈

创建仓库

创建存储区

填写信息

创建仓库

创建本地仓库

填写信息

创建代理仓库( 默认会先从本地仓库搜索jar包，没有的话就会从代理库中下载)

填写信息


阿里云的url：http://maven.aliyun.com/nexus/content/groups/public/

创建仓库组 （可将多个仓库配置到当中）

填写信息，并把刚刚创建的两个库放入其中

至此你的私服就搭建成功啦，下面教大家怎么上传jar包到私服 以及从私服中下载jar包把

上传jar包到私服中

在setting.xml文件中配置 私服 账号信息

在项目的pom文件中配置仓库信息 id 需要与 setting.xml 中的<server> 的 id 匹配哦

pom.xml 编写 包名 以及 版本信息

然后在idea中deploy 就可以了

随后回到私服的网页可视化工具中

看到了 你刚上传的jar包 就证明 你成功啦 😎~~

查看自己上传的jar包的 maven 代码

先修改setting.xml 文件 将地址换成你的私服地址

然后去找到自己的jar包 点进去

再点击

选择Usage 就能看到你要的代码啦~
