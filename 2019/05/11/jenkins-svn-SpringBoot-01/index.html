Jenkins是什么鬼？

Jenkins是一个开源软件项目，是基于Java开发的一种持续集成工具，用于监控持续重复的工作，旨在提供一个开放易用的软件平台，使软件的持续集成变成可能。
—— 百度词条

在我理解就是一个帮助开发人员部署项目的老头子管家。其提供数百个插件用于 构建，部署，自动化 任何项目! 有了jenkins 就可以告别繁琐的打包，上传，重启项目 … 一系列无脑且繁琐的工作，让开发人员更加关注于业务本事。

Jenkins 下载 及 运行

jenkins 是使用JAVA 语言开发的，首先保证系统中已经安装了jdk，最好是jdk1.5以上。（ 我的环境是 jdk1.8 ）

jenkins 下载地址 ： https://jenkins.io/download/

jenkins官方提供了各种版本供客官选择，这里我下载的是 2.174版本的war包 。


war包 只需要放入tomcat 的 webapps 中 然后启动 tomcat 即可。也可以再Jenkins下载目录，打开cmd，运行
java -jar jenkins.war --httpPort=8080

Jenkins 初次运行 创建账号，及安装插件

第一次启动jenkins 需要输入密码 按照页面提示找到密码 输入即可


初始化过程就不多讲，账号可以初始化的时候创建，也可以进入系统后再创建。

安装插件

安装 Publish Over SSH 插件：
进入系统管理 –> 插件管理 –> 可选插件 搜索 Publish Over SSH 然后 点击直接安装即可

配置ssh server

进入 系统管理–> 系统设置 –> 找到 ssh servers

最后点击保存即可
